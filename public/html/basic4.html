テストと品質管理

コードの品質を保つためのテスト手法とデバッグ方法を学びます。

1. 単体テスト

関数が正しく動作するかをテストする方法を学びます。

function add(a, b) {
    return a + b;
}

const test1 = add(2, 3) === 5 ? "成功" : "失敗";
const test2 = add(0, 0) === 0 ? "成功" : "失敗";
console.log("テスト1:", test1);
console.log("テスト2:", test2);

2. デバッグ手法

コードの動作を確認するためのデバッグ方法を学びます。

function calculate(x, y) {
    console.log("入力値:", x, y);
    let result = x * y + 10;
    console.log("計算結果:", result);
    return result;
}

calculate(5, 3);

3. エラーハンドリング

エラーが発生した場合の適切な処理方法を学びます。

try {
    let result = 10 / 0;
    console.log("結果:", result);
} catch (error) {
    console.log("エラーが発生しました");
}

4. パフォーマンス測定

コードの実行時間を測定する方法を学びます。

const startTime = performance.now();
let sum = 0;
for (let i = 0; i < 1000; i++) {
    sum += i;
}
const endTime = performance.now();
console.log("実行時間:", (endTime - startTime).toFixed(2) + "ms");
