JavaScript

JavaScriptとは？

JavaScript（ジャバスクリプト）は、主にWebページに動きをつけたり、ユーザーの操作に応じて内容を変化させたりするために使われるプログラミング言語です。ブラウザ上で動作し、Web開発において欠かせない存在です。

1. 変数とデータ型

JavaScriptの基本的なデータ型と変数宣言方法を学びます。

let age = 25;
let name = "田中太郎";
let isStudent = true;

console.log("年齢:", age);
console.log("名前:", name);

2. 演算子

算術・比較・論理演算子の使い方を学びます。

let x = 10, y = 3;
console.log("足し算:", x + y);
console.log("比較:", x > y);
console.log("論理:", x > 5 && y < 5);

3. 制御構文

if文やforループなどの制御構文を学びます。

let score = 85;
if (score >= 90) {
    console.log("優秀！");
} else {
    console.log("良い！");
}

4. 関数

関数の定義と呼び出し、アロー関数について学びます。

function greet(name) {
    return `こんにちは、${name}さん！`;
}

console.log(greet("山田"));

5. オブジェクトと配列

オブジェクトや配列の基本操作を学びます。

let student = { name: "鈴木花子", age: 20 };
let fruits = ["りんご", "バナナ"];

console.log("学生:", student.name);
console.log("果物:", fruits[0]);

6. 配列の便利なメソッド

配列の主な操作（map, filter）を学びます。

let numbers = [1, 2, 3, 4, 5];
let doubled = numbers.map(num => num * 2);
let evens = numbers.filter(num => num % 2 === 0);

console.log("2倍:", doubled);
console.log("偶数:", evens);

7. DOM操作の基礎

HTML要素を取得して内容を変更する方法を学びます。

let target = document.getElementById('demo');
target.textContent = "変更されました！";
target.style.color = "blue";

console.log("要素を変更しました");
ここが変わります

8. テンプレートリテラル

文字列の中に変数を埋め込む便利な方法を学びます。

let userName = "田中";
let userAge = 25;
let message = `ユーザー: ${userName}, 年齢: ${userAge}歳`;

console.log(message);
