セキュリティと実践パターン

実際の開発で重要なセキュリティ対策と実践的なパターンを学びます。

1. XSS対策

クロスサイトスクリプティング攻撃を防ぐ方法を学びます。

function sanitizeInput(input) {
    return input.replace(//g, '>');
}

const userInput = "<script>alert('悪意のあるコード')</script>";
const safe = sanitizeInput(userInput);
console.log("安全な文字列:", safe);

2. データ検証

ユーザー入力の検証パターンを学びます。

function validateEmail(email) {
    const pattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return pattern.test(email);
}

console.log("有効:", validateEmail("test@example.com"));
console.log("無効:", validateEmail("invalid-email"));

3. パフォーマンス最適化

効率的なコードの書き方を学びます。

console.time("処理時間");
let sum = 0;
for (let i = 0; i < 1000; i++) {
    sum += i;
}
console.timeEnd("処理時間");
console.log("合計:", sum);
