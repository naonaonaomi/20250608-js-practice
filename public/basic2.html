7. クラスとインスタンス

JavaScriptのclass構文とインスタンス生成

class Animal {
  constructor(name) {
    this.name = name;
  }
  greet() {
    return `こんにちは、${this.name}です`;
  }
}
const dog = new Animal('ポチ');
dog.greet();
