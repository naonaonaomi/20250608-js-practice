JavaScript

JavaScriptとは？

JavaScript（ジャバスクリプト）は、主にWebページに動きをつけたり、ユーザーの操作に応じて内容を変化させたりするために使われるプログラミング言語です。ブラウザ上で動作し、Web開発において欠かせない存在です。

1. 変数とデータ型

JavaScriptの基本的なデータ型と変数宣言方法を学びます。

let num = 10;
const str = "こんにちは";
let flag = true;
let obj = { name: "Taro" };
let arr = [1, 2, 3];

2. 演算子

算術・比較・論理演算子の使い方を学びます。

let a = 5;
let b = 3;
let sum = a + b;
let isEqual = (a === b);
let logic = (a > 2 && b < 5);

3. 制御構文

if文やforループなどの制御構文を学びます。

let msg = "";
for(let i=1; i<=3; i++) {
  if(i % 2 === 0) {
    msg += i + "は偶数\n";
  } else {
    msg += i + "は奇数\n";
  }
}

4. 関数

関数の定義と呼び出し、アロー関数について学びます。

function greet(name) {
  return "こんにちは、" + name + "さん";
}
const arrow = (x) => x * 2;

5. オブジェクトと配列

オブジェクトや配列の基本操作を学びます。

let user = { name: "Hanako", age: 20 };
let nums = [10, 20, 30];
nums.push(40);
let names = ["A", "B", "C"];
let upper = names.map(n => n.toUpperCase());

5.5 配列操作

配列の主な操作（push, pop, shift, unshift, splice, map, filter, find）を学びます。

let arr = [1, 2, 3, 4, 5];
arr.push(6);      // [1,2,3,4,5,6]
arr.pop();        // [1,2,3,4,5]
arr.shift();      // [2,3,4,5]
arr.unshift(0);   // [0,2,3,4,5]
arr.splice(2,1,99); // [1,2,99,4,5]
arr.map(x => x*2); // [2,4,6,8,10]
arr.filter(x => x%2===0); // [2,4]
arr.find(x => x>3); // 4

ボタンを押すとテキストが変わるなど、DOM操作を体験します。

7. エラー処理

try...catchによるエラー処理を学びます。

try {
  throw new Error("エラー発生！");
} catch (e) {
  console.log(e.message);
}

テンプレートリテラルや分割代入、スプレッド構文などを学びます。
